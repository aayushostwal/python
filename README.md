## Getting Started

### Prerequisites
- Python 3.12+ (examples use modern Python features)
- Jupyter Notebook or JupyterLab

### Running the Notebooks

```bash
# Clone the repository
git clone <repo-url>
cd python

# Install Jupyter (if not installed)
pip install jupyter

# Launch Jupyter
jupyter notebook
```

## Course Structure

Each module is a self-contained Jupyter notebook with:
- **Theory sections** explaining core concepts
- **Code examples** you can run and modify
- **Practical exercises** to reinforce learning
